
Guide to execute fhir-stream-writer and fhir-stream-feeder tool 
----------------------------------------------------------------

1) unzip frt-service-package
   1.1) unzip frt-service-package-1.0.0-SNAPSHOT.zip to /home/ec2-user/frt-service-package
   1.2)	chmod +x to all the *.sh if necessary.

2) start confluent/fakfa

3) run fhir-stream-writer tool

   Data flow: read patient records in json file from data folder and publish to FhirTopic
   
   3.1) cd /home/ec2-user/frt-service-package/bin
   3.2) run ./fhir-stream-writer.sh ../data 
        where ../data folder contains patient json files
		
4) run fhir-stream-feeder tool 

   Data flow: generate patient records, extract the generated patient records, and publish to FhirTopic

   4.1) cd /home/ec2-user/frt-service-package/bin
   4.2) run ./fhir-stream-feeder.sh ../data 100 California 10 Arizona 20 Virginia 30
		where ../data folder -- is the feeder intermediate files stagging area(it is created if not exists)

			  100 -- time interval in milli-sec bewteem two messages delivered
			  California 10 Arizona 20 Virginia 30 -- list of  a pair of state name and sample size 
			  
			  ../data folder 
				-- contains a list of state folders, each contains patient json messages 
				   in that state generated by synthea 
		
				 
			  ../data/extracted folder 
			    -- contains a list of state folders, each contains patient json messages 
				   in that state generated by extractor tool using patien json messages created by by synthea
